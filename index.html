OmniAuth Shopify

Shopify OAuth2战略OmniAuth 1.0。

#安装

添加到您的“Gemfile”:

“‘红宝石 宝石的omniauth-shopify-oauth2 “””

然后包安装的。

#使用

“OmniAuth::策略::Shopify”只是一个架中间件。 读(OmniAuth 1.0文档)(https://github.com/intridea/omniauth)的详细说明。

这里有一个例子,一个Rails应用的中间件的配置初始化/ omniauth.rb:

“‘红宝石 Rails.application.config.middleware。 使用OmniAuth::Builder 提供者:shopify ENV(“SHOPIFY_API_KEY”),环境(“SHOPIFY_SHARED_SECRET”) 结束 “””

#配置

您可以配置范围,您通过“提供者”方法通过散列:

*“范围”:一个以逗号分隔的权限你想要从用户的请求。 看到(Shopify API文档)(http://docs.shopify.com/api/tutorials/oauth)的完整列表可用的权限。

例如,请求‘read_products’,‘read_orders’和‘write_content权限和显示身份验证页面:

“‘红宝石 Rails.application.config.middleware。 使用OmniAuth::Builder 提供者:shopify ENV[' SHOPIFY_API_KEY '],ENV(“SHOPIFY_SHARED_SECRET”):范围= > ' read_products、read_orders write_content” 结束 “””

#验证散列

这里有一个例子认证散列中可用“request.env[' omniauth.auth ']':

“‘红宝石 { :供应商= > ' shopify ', :uid = > ' example.myshopify.com ', :凭证= > { :令牌= > afasd923kjh0934kf,# OAuth 2.0 access_token,你存储和使用API请求进行身份验证 } } “””
